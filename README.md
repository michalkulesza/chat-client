## Tech (front & back)

- React
- TypeScript
- NodeJS
- Sass
- Socket.io
- Mongoose (MongoDB)
- REST
- Hooks
- Heroku

## Features

- Login users with validation
- Form Validation
- Chat action messages
- Search active users
- Start a private chat with a user
- Data persistency with MongoDB
